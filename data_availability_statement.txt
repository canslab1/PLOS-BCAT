Data Availability Statement
(For Editorial Manager submission form)

All simulation source code (NetLogo 4.0.5 and Python 3 reimplementation) and parameter configuration files are publicly available without restriction under the MIT License at https://github.com/canslab1/BCAT, with a permanent archived copy deposited on Zenodo (DOI: 10.5281/zenodo.19081523). The raw simulation output data underlying the tables and figures in this study are available at https://github.com/canslab1/BCAT/tree/main/data, including: (1) sensitivity analysis results across four network topologies (Table 3 and Figs 7-9), (2) mechanism decomposition experiment results (Fig 10, 30,000 runs), and (3) finite-size scaling experiment results (N = 400 to 2,500). Analysis scripts for reproducing all reported results are provided at https://github.com/canslab1/BCAT/tree/main/scripts.
